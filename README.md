# Advent Of Code 2018

Doing the Advent of Code 2018 challenges in Haskell.

## Using

### Stack

```shell
> stack build
> stack repl adventofcode2018:lib
> stack run day-01
> stack test :test-01
```

### Cabal

```shell
> cabal build
> cabal repl adventofcode2018
> cabal run day-01
> cabal test test-01
```
